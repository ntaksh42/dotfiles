# openfolder

現在の作業ディレクトリ（または引数で指定されたパス）を Windows エクスプローラーで開く。

## ワークフロー

1. **開くパスを決める**
   - 引数（`$ARGUMENTS`）が指定されていればそのパスを使う。
   - 指定がなければ、PowerShell ツールで `$PWD.Path` を実行して現在の cwd を取得する。

2. **エクスプローラーで開く**
   PowerShell ツールで以下を実行する（パスは必ずダブルクォートで囲む）:
   ```powershell
   Invoke-Item "<パス>"
   ```
   - パスが Windows 形式（`C:\Users\...`）であることを確認する。Bash の `pwd` を使った場合は `/c/Users/...` 形式になるので、PowerShell の `$PWD.Path` を使うか Windows 形式に変換する。

3. **報告**
   開いたフォルダのパスを一行で報告する。

## 注意

- `Invoke-Item` はフォルダを開いた後すぐに制御を返す。GUI を待たない。
- 指定パスが存在しない場合はエラーになるので、その旨をユーザーに伝える。
